- The main form for Id and Name act as a search form too.
- If a prompted Id matches a record in the list, the name box will be automatically filled with the matching name, and vice-versa.
- Is possible to modify the previously entered Id and Name that matched using the same text boxes, change student type, and select eventual dorm and meal plan. 
- Clicking on Update Student it will automatically replace the record that matched the research with the new data. Or you can create a new one clicking on Add Student 
   if the Id has been modified, and is not required to overwrite the selected object from before.
- Double IDs are not allowed.
